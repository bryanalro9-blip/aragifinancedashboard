🔗 Koneksi Google Sheets

Data tersimpan permanen di Google Sheets milik kamu — aman, bisa diakses dari mana saja, bertahun-tahun.

Paste URL dari Apps Script deployment kamu di sini.

Cara Setup (5 menit, sekali selamanya)

1
Download file Apps Script

Buka file aragi_appscript.gs yang sudah dibuatkan — isinya adalah kode backend untuk Google Sheets kamu.

2
Buka Google Sheets baru

Buat spreadsheet baru di sheets.google.com. Kasih nama "Aragi Finance DB" atau apapun.

3
Buka Apps Script

Di Sheets: klik menu Extensions → Apps Script. Hapus semua kode yang ada, paste seluruh isi file .gs.

4
Deploy sebagai Web App

Klik Deploy → New deployment. Type: Web app. Execute as: Me. Who has access: Anyone. Klik Deploy.

5
Copy URL & paste di sini

Copy URL yang muncul (bentuknya: https://script.google.com/macros/s/...), paste di field URL di atas, klik Hubungkan.
